# SHUO-UI

A Component Library for Vue3.js.
Doc address: https://www.shuoui.com

## Install

```bash
npm install shuo-ui --save
```

## How to use

引入 css

```js
// main.ts
import 'shuo-ui/dist/style.css'
```

页面

```vue
<script setup>
import { SButton } from 'shuo-ui'
</script>

<template>
  <s-button>按钮</s-button>
</template>
```
